## Installation and Running

### Development

```bash
# Install dependencies
npm install

# Start dev server
npm start
# or
npm run dev
```

The app will be available at `http://localhost:5173`

### Build
```bash
# Install dependencies
npm install

# Build for production
npm run build
```

Built files will be in the `dist/` folder
